# DIT Campus Map (PWA)

Smart Campus Navigation PWA scaffold for DIT University, Dehradun.

## Setup

```bash
npm install
npm run dev
```

## Notes

- PWA manifest: `public/manifest.json`
- PWA plugin config: `vite.config.js`
- Firebase placeholders: `src/services/firebase.js`
- Zustand store: `src/store/mapStore.js`
